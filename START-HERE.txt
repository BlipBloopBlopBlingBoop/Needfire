NEEDFIRE — offline survival computer
=====================================

Everything runs on THIS computer. No internet, no account, no cost.


READ THIS FIRST
  Needfire is a reference library — not a doctor and not an emergency
  service. If a phone works, call your local emergency number (911 / 112)
  FIRST. Use at your own risk; the plain-language details are in
  DISCLAIMER.md (any text editor opens it).


START IT — double-click the launcher for your system
  Windows   Start-Needfire-Windows.bat
  Mac       Start-Needfire-Mac.command   (the FIRST time: right-click → Open)
  Linux     Start-Needfire-Linux.sh

  Your browser opens the app by itself at  http://localhost:8848
  The first start takes about a minute (it builds the library index).
  To stop: close the window, or press Ctrl+C inside it.
  If it says Python is missing, the launcher opens the download page —
  install Python, then double-click the launcher again.


THE FULL GUIDE — QUICKSTART.md (any text editor opens it)
  Everything else, step by step: using Needfire from your phone and
  installing it as an app so Emergency mode works with no signal; running
  it right on an iPhone or iPad; turning a spare computer into a dedicated
  always-on "Bothy" appliance; and fixes for common problems.


YOUR DATA
  Downloads and the search index live in the hidden ".needfire-home"
  folder inside this folder. Copy this whole folder to back everything up.
